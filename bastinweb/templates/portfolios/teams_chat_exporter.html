Teams Chat Exporter

Languages/Technologies:
Python
Flask
Bootstrap 5
Microsoft Graph API

Problem Statement

Internal policies dictated that employees must be able to export their chat logs from Microsoft Teams for compliance and record-keeping purposes. However, Microsoft Teams did not provide a straightforward way for end users to export their chat data in a user-friendly format.

Project Description

This Teams Chat Exporter is a web application developed using Python and Flask that allows end users to easily export their chat logs from Microsoft Teams. The application authenticates via an Entra application and uses the Microsoft Graph API to retrieve their chat data. Users simply copy the conversation link from Teams, paste it into the web app, and export them as PDFs.
